# CALCULTOR
# Simple C Calculator

This is a basic menu-driven calculator application developed in C. It allows users to perform standard arithmetic operations through a simple command-line interface.

## 🚀 Features
- **Addition (+)**
- **Subtraction (-)**
- **Multiplication (*)**
- **Division (/)**
- **User-friendly Menu**: Easy navigation with numbered choices.
- **Exit Option**: Gracefully exit the program using the menu.

## 🛠️ How to Run
To compile and run this program on your local machine, ensure you have a C compiler (like GCC) installed.  
# for turmux ,terminal and vs 

$ pkg install clang  
  



1. **Clone the repository:**
   ```bash
   git clone https://github.com/Devprinceee/CALCULTOR
   ```
2. **Navigate to the directory:**
   ```bash
   cd CALCULTOR
   ```
3. **Compile the code:**
   ```bash
   gcc CALCULATOR.c -o CALCULTOR
   ```
   **for termux;**
   
   $ clang CALCULTOR.c -o CALCULTOR
5. **Run the executable:**
   ```bash
   ./calculator
   ```

## 📝 Code Structure
The program uses a `while` loop to keep the menu active until the user chooses to exit. It utilizes `scanf` for input and `if-else` or `switch` logic (as per your implementation) to handle operations.
